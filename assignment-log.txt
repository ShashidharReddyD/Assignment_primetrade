Project Overview
----------------
This project implements a secure and scalable REST API using Node.js and Express.
The system includes authentication using JWT, role-based access control, and CRUD
operations for managing tasks. A simple frontend UI was also created using Vanilla
JavaScript to interact with the APIs.
